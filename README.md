# Linda Mhlangu Portfolio

Personal portfolio website for **Linda Mhlangu**.

This site is published from the **LinMhlaDoRh.github.io** GitHub Pages repository. `LinMhlaDoRh` is the GitHub username/repo owner, and **Linda Mhlangu** is the person the portfolio represents.

## Sections
- Home: focused junior developer hero with project, CV, GitHub, and LinkedIn actions
- About: background, current focus, and developer strengths
- Skills: back-end, front-end, database, security, tools, and foundations
- Projects: featured project hierarchy with Student Events Platform first
- Education: degree and certifications
- Contact: email, phone, LinkedIn, GitHub, and mailto form

## Deploy
Push these files to the root of the `LinMhlaDoRh.github.io` repository on the `main` branch.
